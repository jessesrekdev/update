HTML Attributes

Attributes provide extra information about HTML elements — from href and src to alt, title and lang. Learn how to use them correctly and consistently.

HTML Attributes Overview

Attributes are specified in the start tag and usually come in name/value pairs like name="value".

  • ✓ All HTML elements can have attributes
  • ✓ Attributes provide additional information about elements
  • ✓ Attributes are always specified in the start tag
  • ✓ Attributes usually come in name/value pairs like: name="value"
<img srcSpecifies the image source="image.jpg" altProvides alternative text="Description" widthSets the image width="300">

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.

<a href="https://example.com">Visit Example</a>

You will learn more about links in our HTML Links chapter.

The src Attribute

The <img> tag is used to embed an image. The src attribute specifies the path to the image.

External image (hosted elsewhere):

<img src="https://example.com/img_girl.jpg">
Notes: External images may be removed or copyrighted.

Image hosted within site:

<img src="img_girl.jpg">
Tip: Prefer relative URLs for site content.

Common Image Attributes

width & height

<img src="img_girl.jpg" width="500" height="600">

alt

<img src="img_girl.jpg" alt="Girl with a jacket">

Best Practices

Always Use Lowercase Attributes

W3C recommends lowercase attribute names in HTML for consistency.

<a href="...">

Always Quote Attribute Values

Use quotes around attribute values. When values contain quotes, use the opposite type.

<a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>
<a href='https://www.w3schools.com/html/'>Visit our HTML tutorial</a>
<a href=https://www.w3schools.com/html/>Visit our HTML tutorial</a>
Warning: No quotes can break when values contain spaces.
