HTML Attributes

HTML attributes provide additional information about HTML elements, enhancing their functionality and behavior.

HTML Attributes

  • ✓ All HTML elements can have attributes
  • ✓ Attributes provide additional information about elements
  • ✓ Attributes are always specified in the start tag
  • ✓ Attributes usually come in name/value pairs like: name="value"
<img srcSpecifies the image source="image.jpg" altProvides alternative text="Description" widthSets the image width="300">

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

<a href="https://example.com">Visit Example</a>

You will learn more about links in our HTML Links chapter.

The src Attribute

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:

<img src="img_girl.jpg">

URL Types

Absolute URL
Relative URL

Links to an external image that is hosted on another website:

<img src="https://example.com/img_girl.jpg">

Notes: External images might be under copyright. If you do not get permission to use it, you may be in violation of copyright laws. In addition, you cannot control external images; they can suddenly be removed or changed.

Links to an image that is hosted within the website:

<img src="img_girl.jpg">

Tip: It is almost always best to use relative URLs. They will not break if you change domain.

Common Image Attributes

width & height

Specifies the dimensions of the image (in pixels):

<img src="img_girl.jpg" width="500" height="600">

alt Attribute

Specifies an alternate text for an image:

<img src="img_girl.jpg" alt="Girl with a jacket">

Other Important Attributes

The style Attribute

Used to add styles to an element:

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

Declares the language of the Web page:

<!DOCTYPE html>

<html lang="en">

<body>

</body>

</html>

The title Attribute

Defines extra information shown as a tooltip:

<p title="I'm a tooltip">This is a paragraph.</p>

Best Practices

Always Use Lowercase Attributes

The HTML standard does not require lowercase attribute names, but W3C recommends lowercase attributes in HTML.

Recommended

<a href="...">


Not Recommended

<a HREF="...">

Always Quote Attribute Values

The HTML standard does not require quotes around attribute values, but W3C recommends quotes in HTML.

Double Quotes
Single Quotes
No Quotes

<a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>

<a href='https://www.w3schools.com/html/'>Visit our HTML tutorial</a>

<a href=https://www.w3schools.com/html/>Visit our HTML tutorial</a>

Warning: This may cause issues when attribute values contain spaces.

When attribute values contain quotes, use the opposite type:

<p title='John "ShotGun" Nelson'>

<p title="John 'ShotGun' Nelson">
