HTML Headings

HTML headings are defined with the <h1> to <h6> tags.

<h1> defines the most important heading. <h6> defines the least important heading.

<h1>Heading 1</h1>

<h2>Heading 2</h2>

<h3>Heading 3</h3>

<h4>Heading 4</h4>

<h5>Heading 5</h5>

<h6>Heading 6</h6>

Note: Browsers automatically add some white space (a margin) before and after a heading.

Why Headings Are Important

SEO Optimization

Search engines use headings to index your content structure.

User Experience

Readers skim content using headings for better navigation.

Accessibility

Screen readers use headings to help visually impaired users.

<h1> headings should be used for main headings, followed by <h2> headings, then the less important <h3>, and so on.

Warning: Use HTML headings for headings only. Don't use headings to make text BIG or bold. Use CSS for styling instead.

Customizing Heading Sizes

Each HTML heading has a default size. However, you can specify the size for any heading with the style attribute, using the CSS font-size property:

<h1 style="font-size:60px;">Heading 1</h1>

Best Practices for Heading Sizes:

  • Maintain a clear hierarchy (h1 > h2 > h3, etc.)
  • Use relative units (em, rem) for responsive designs
  • Ensure sufficient contrast with the background
  • Keep line height proportional to font size
